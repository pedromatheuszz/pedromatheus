# Portfólio Profissional

Site estático criado para divulgar o trabalho de Pedro Matheus como programador.

## Arquivos principais

- `index.html`: estrutura, conteúdo e seções do portfólio
- `styles.css`: identidade visual, responsividade e animações
- `script.js`: dados principais do perfil, menu mobile e interações
- `favicon.svg`: ícone do site

## O que editar primeiro

- Atualize nome, cargo, e-mail, GitHub e LinkedIn no objeto `portfolioData` em `script.js`
- Ajuste os textos principais de apresentação em `index.html`
- Personalize a seção `#solucoes` com seus trabalhos ou tipos de projeto

## Como abrir

Abra o arquivo `index.html` no navegador.

## Publicação

Você pode publicar este site facilmente em serviços como GitHub Pages, Netlify ou Vercel.
